# Notas

Uma possibilidade para usar o microcontrolador ATMega328P da ATMEL é usar o conjunto de ferramentas de desenvolvimento da MicroChip para escrever o assembly da arquitetura AVR diretamente para o chip. Sem um hardware programador dedicado precisamos usar um pacote chamado AVRDUDE que nos permite escrever para a flash via USB, esse pacote vem incluso na distribuição Arduino IDE já que é a ferramenta que o grupo usa para programar o kit de desenvolvimento arduino via USB.
